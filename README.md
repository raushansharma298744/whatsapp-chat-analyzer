## 📊 WhatsApp Chat Analyzer

This project is a data analysis tool that processes exported WhatsApp chat files and generates meaningful insights and visualizations.

### 🚀 Features
- 📈 Message statistics (total messages, words, media, links)
- 👤 Most active users analysis (for group chats)
- 🕒 Timeline analysis (daily & monthly activity)
- 🔤 Word frequency & most common words
- 😂 Emoji analysis
- 🔥 Activity heatmaps
- 📊 Data visualization using graphs

### 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib / Seaborn
- Streamlit (if used)

### 📂 How to Use
1. Export WhatsApp chat (without media)
2. Upload the `.txt` file into the app
3. Get insights instantly

### 🎯 Use Cases
- Understand chat behavior
- Analyze group activity
- Personal chat insights

### 📌 Future Improvements
- Sentiment analysis
- NLP-based insights
- Chat summarization
